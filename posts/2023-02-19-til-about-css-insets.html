TIL about CSS Insets

Feb 19 2023

Just a quick tip that I thought is worth sharing. Instead of declaring:

.foo { 
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
}

you can just use:

.foo {
  inset: 0;
}

It's supported everywhere computers are sold!

MDN: https://developer.mozilla.org/en-US/docs/Web/CSS/inset
